# Massing Generator

AI-генератор массинга зданий: DWG на вход, DWG на выход.

## Что делает

Автоматически генерирует оптимальную расстановку зданий (массинг) на пятне застройки с учётом строительных норм и заданных параметров.

## Пайплайн

```
DWG → (ODA Converter) → DXF → парсинг → AI генерация → запись DXF → (ODA Converter) → DWG
```

1. **Конвертация** — DWG→DXF через ODA File Converter
2. **Парсинг** — извлечение границ пятна застройки из DXF (ezdxf)
3. **Генерация** — AI генерирует массинг с учётом норм (Claude API)
4. **Валидация** — проверка плотности, отступов, инсоляции
5. **Запись** — результат в DXF с объёмами зданий
6. **Конвертация** — DXF→DWG через ODA File Converter

## Входные параметры

| Параметр | Описание | Пример |
|----------|----------|--------|
| `input` | DWG/DXF файл с пятном застройки | `site.dwg` |
| `density` | Макс. плотность застройки | `2.5` |
| `target_area` | Целевая продаваемая площадь, м² | `45000` |
| `region` | Регион (для норм) | `moscow` |

## Нормативная база

- СП 42.13330 (Градостроительство)
- СанПиН по инсоляции
- Противопожарные нормы (разрывы между зданиями)
- Региональные ПЗЗ

## Зависимости

- Python 3.10+
- [ODA File Converter](https://www.opendesign.com/guestfiles/oda_file_converter) (для DWG↔DXF)
- Пакеты: см. `requirements.txt`

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python massing.py --input site.dwg --density 2.5 --target-area 45000 --region moscow
```
